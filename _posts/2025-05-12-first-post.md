Agar aap ek developer, blogger, ya student hain, to aapke liye **GitHub Pages** aur **Jekyll** ek powerful combination ho sakta hai free website banane ke liye.

## Step 1: GitHub Account Banaye

Sabse pehle [GitHub.com](https://github.com) par apna free account banaye.

## Step 2: New Repository Banaye

- Repository ka naam rakhein: `username.github.io`
- Initialize with README (optional)

## Step 3: GitHub Pages Enable Karein

- Go to `Settings > Pages`
- Source select karein: `main` branch and `/ (root)`

## Step 4: Jekyll Setup Karein

1. `_config.yml` file banayein
2. `index.md` aur `_posts/` folder add karein
3. Layouts aur includes customize karein

## SEO Tips:

- Har post mein unique title aur description likhein
- `jekyll-seo-tag` plugin use karein
- Clean URLs rakhein (permalink: `/:title/`)

## Conclusion

Ab aapki website live ho chuki hai! Apna domain bhi connect kar sakte hain.

**Aapka free aur SEO friendly blog ready hai — aur wo bhi bina hosting charges ke!**
